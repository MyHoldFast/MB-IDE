Дополн. функции

FRE(M%) — возвращает статистику памяти (в байтах) в зависимости от значения M%:
0 - память оставшаяся для программы
1 - размер текущей программы
2 - максимум байт, доступных для программы
3 - количество свободной памяти телефона
4 - вся память телефона

ERR(0) — в случае ошибки возвращает её код

READDIR$(M$) — возвращает результат поиска файла по маске: если найдено, то имя файла, если нет, то "". Пример маски: "c*","*.txt","w*.dat"

PROPERTY$(M$) — возвращает параметры телефона по таким аргументам:
microedition.model
microedition.profile
microedition.configuration
microedition.locale
microedition.encoding

NUMCOLORS(0) — количество цветов, отображаемые телефоном

ISCOLOR(0) — если экран телефона цветной, возвращает 1, иначе - 0
