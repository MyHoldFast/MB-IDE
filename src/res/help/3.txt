Функции пользовательского интерфейса

SELECT(T$,P1$,P2$,...Pn$) [сокращенная форма SEL]
Меню выбора, где:
T$ — заголовок
P1$,P2$...,Pn$ — пункты меню
Возвращает номер выбранного пункта.
ALERT T$,txt$,gel$,type%,timeout% [сокращенная форма AL]
Окно сообщения, где:
T$ — заголовок
txt$ — сообщение
gel$ — граф. элемент
type% — тип сообщения:
0 — Confirmation(подтверждение)
1 — Info(информация)
2 — Warning(предупреждение)
3 — Error(ошибка)
4 — Alarm(напоминание)
timeout% - время демонстрации Alert в MS(0 - неограниченно!).
(пока отображен Alert - программа продолжает работать на заднем плане!)
 
MESSAGEFORM(T$,O$,C$,L$,M$):[сокращенная форма MSF]
Выводит на экран текст.
Возвращает 1 - если "ок", -1 если "отмена".
где:
T$ - заголовок
O$ - надпись на клавише "ок"
C$ - надпись на клавише "отмена"
L$ - подзаголовок
M$ - сообщение.

EDITFORM(T$,O$,C$,L$,V$,L%,M%)[сокращенная форма EF]
Редактирование текста.
Возвращет: len(V$) - "ок", -1 - "отмена"
V$ - переменная редактирования
L% - максимальная длина
M% - тип:
0 - текст
1 - пароль
2 - число
3 - e-mail
4 - номер телефона
5 - url

CHOICEFORM(T$,O$,C$,L$,A$,M%)[сокращенная форма CF]
Выбор.
Возвращает: N - "ок", -1 - "отмена"
A$ - массив выбора
M% - тип:
0 - переключатели
1 - флажки

GAUGEFORM(T$,O$,C$,L$,M%,S%,M%)[сокращенная форма GF]
Настройка уровня.
Возвращает: N - "ок", -1 - "отмена"
M% - размер
S% - стартов. значение
M% - тип:
0 - нерегулируемый
1 - регулируемый

DATEFORM(T$,O$,C$,L$,A%,M%)[сокращенная форма DF]
Настройка даты
Возвращает: 1 - "ок", -1 - "отмена"
A% - массив
M% - тип:
0 - время и дата
1 - дата
2 - время
A%(0)=MS%
A%(1)=D%
